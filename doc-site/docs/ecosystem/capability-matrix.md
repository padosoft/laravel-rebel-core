# Capability Matrix

| Capability | Primary packages |
|---|---|
| Shared contracts and assurance | `laravel-rebel-core` |
| Passwordless email OTP | `laravel-rebel-email-otp` |
| Per-action step-up | `laravel-rebel-step-up` plus bridges |
| Fortify integration | `laravel-rebel-bridge-fortify` |
| Passkey integration | `laravel-rebel-bridge-passkeys` |
| TOTP integration | `laravel-rebel-bridge-laragear-2fa`, `laravel-rebel-bridge-spatie-otp` |
| OTPZ integration | `laravel-rebel-bridge-otpz` |
| SMS/WhatsApp/voice channel abstraction | `laravel-rebel-channels` |
| Provider delivery | Twilio, Vonage, Bird, Telegram, Discord packages |
| Session/device governance | `laravel-rebel-sessions` |
| Account recovery | `laravel-rebel-recovery` |
| Bot protection | `laravel-rebel-bot-protection` |
| Control plane API/UI | `laravel-rebel-admin-api`, `laravel-rebel-admin` |
| AI anomaly assistant | `laravel-rebel-ai-guard` |
